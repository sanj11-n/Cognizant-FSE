# Week 1: Design Patterns and Principles - Mandatory Hands-on

This repository contains all the mandatory hands-on assignments for Week 1, organized cleanly into structured Java packages.

## Projects Structure
* `exercise1` - Singleton Pattern (Logger Implementation)
* `exercise2` - Factory Method Pattern (Document Creation System)
* `exercise3` - Builder Pattern (Step-by-step Computer Configuration)
* `exercise4` - Adapter Pattern (Legacy System Integration Translator)
* `exercise5` - Decorator Pattern (Dynamic Notification Extender)
* `exercise6` - Observer Pattern (News Agency Event Subscription Alert)
* `exercise7` - Strategy Pattern (Runtime Interchangeable Payment System)
* `exercise8` - Command Pattern (Decoupled Light/Remote Action Request)

## How To Run
1. Open the project folder in IntelliJ IDEA.
2. Navigate to `src/exercise1/Main.java`.
3. Click the green play icon next to the main method to run all exercises simultaneously.
