# Cognizant FSE Training - Week 3: Spring Framework Core

This repository contains the complete unified hands-on implementation tracking for Week 3, focusing on core Spring architecture design principles.

---

## 🛠️ Core Deliverables Implemented

### 1. Dependency Injection (`NotificationSystem.java`)
* Loose coupling setup utilizing constructor injection patterns to seamlessly swap messaging service implementations.

### 2. Inversion of Control Container (`SpringIoCContainer.java`)
* Custom application context component simulation managing custom bean instantiation and container registry lookups.

### 3. Aspect-Oriented Programming (`AOPLoggingSystem.java`)
* Proxy pattern implementation intercepting business actions to run cross-cutting logging aspects before and after core methods.

### 4. Bean Lifecycle Management (`BeanLifecycleManager.java`)
* Comprehensive execution detailing property injection, custom initialization stages, and container shutdown destruction routines.
